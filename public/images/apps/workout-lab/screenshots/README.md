# Workout Lab Screenshots

Place your Workout Lab app screenshots in this directory.

## Recommended Screenshots (5-6 total)

1. **analysis-charts.png** - Analysis charts showing Max/Avg/Min, dual Y-axis, 1RM evolution
2. **superset-logging.png** - Superset/flow interface showing seamless exercise editing
3. **goal-progress.png** - Goal tracking with delta of improvement, multi-metric view
4. **exercise-variety.png** - Exercise logging showing different types (strength, cardio, etc.)
5. **dark-mode.png** - Dark mode example showcasing the UI
6. **overview.png** (optional) - Main dashboard or overview screen

## Image Guidelines

- **Format**: PNG or WebP (WebP preferred for better performance)
- **Size**: 1080px width recommended (will be optimized for web)
- **Aspect Ratio**: Mobile screenshot ratio (roughly 9:19 or phone native)
- **File Naming**: Use kebab-case (lowercase with hyphens)

## Notes

- Screenshots will be lazy-loaded for performance
- Captions will be added in the gallery component
- You can add screenshots incrementally - placeholders will be used until real images are available
